Group Project 1

This was the first group project of the coding bootcamp from the University of Toronto. The task was to use two APIs to build and populate a site. This site combines the use of a local brewery api, as well as a mapquest api. When a user inputs a city or location, the site will populate the 5 closest breweries to you and display the name, website, address and distance of each. It was a great first project where we didn't really have any guidelines, and had a lot of freedome and creativity options. It was also great to experience what it would be like working in a group environment. This is the github link to access the github repository to all the code for the above project.

Group Project 2

This was the second group project of the coding bootcamp from the University of Toronto. The task was to utilize express, mysql and sequalize to build our own database and create a site for users. The site that we developed was a movie review site, that allows for users to login and leave a review on movies that we have added to our database. There was a lot of troubleshooting and testing to make sure that all the routes we created and forms created all worked. It was great to see a functioning final product and to visually see how far we have come along just past the halfway mark of this bootcamp.
